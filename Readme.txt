🍅Tomato - Full-Stack Food Delivery App (MERN)

Tomato is a complete food delivery platform built with the MERN stack. It features customer-facing ordering, real-time cart updates, Stripe-based payments, and an admin dashboard for managing food items and orders.

🔗Live Demo:https://tomato-frontend-6647.onrender.com/

🚀 Project Highlights:

📌User Authentication:
JWT-based secure login and registration

📌Cart & Orders:
Real-time cart updates and Stripe payment integration

📌Admin Dashboard:
Add/edit/delete food items, update order statuses

📌Responsive Design:
Mobile-first UI with clean and intuitive navigation

⚙️ Tech Stack:

📌Frontend: React (Vite), Context API, Custom CSS

📌Backend: Node.js, Express.js, MongoDB (Mongoose)

📌Integrations: Stripe API (payments), Multer (image uploads)

🌏Hosting (via Render):

 📌Frontend (User site): Static Site

 📌Admin Dashboard: Static Site

 📌Backend/API: Web Service

💡What I Delivered:

📌Developed full-stack features from UI to database

📌Integrated secure payments using Stripe Checkout

📌Managed state efficiently with Context API

📌Built robust RESTful APIs with validation and error handling

📌Handled image uploads for food listings using Multer

🛠️ Local Setup Guide:

1. Clone the repository:

git clone [repo-url]


2. Set Up Environment Variables:

📌Create a .env file in the backend directory:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
STRIPE_SECRET_KEY=your_stripe_secret_key


3. Install & Run Backend:

cd backend
npm install
npm start


4. Install & Run Frontend:

cd frontend
npm install
npm run dev
